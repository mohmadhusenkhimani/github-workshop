# 📘 GitHub Workshop – LJ University

Welcome to the **GitHub Workshop** repository! 🚀

This repository was created as part of the **GitHub Workshop conducted at LJ University**. It serves as a hands-on learning resource for students who are new to Git, GitHub, and version control.

Whether you're uploading your first project or collaborating with others, this workshop covers the essential GitHub workflow.

---

## 🎯 Workshop Objectives

- Understand Version Control
- Learn Git & GitHub fundamentals
- Create and manage GitHub repositories
- Upload projects to GitHub
- Update existing repositories
- Track changes using Git
- Collaborate using branches and Pull Requests
- Host static websites using GitHub Pages

## 🚀 Basic Git Commands

### Initialize Git

```bash
git init
```

### Clone Repository

```bash
git clone <repository-url>
```

### Check Status

```bash
git status
```

### Add Files

```bash
git add .
```

### Commit Changes

```bash
git commit -m "Initial Commit"
```

### Push to GitHub

```bash
git push origin main
```

### Pull Latest Changes

```bash
git pull origin main
```
